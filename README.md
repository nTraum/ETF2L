Ruby wrapper for the [ETF2L API](http://api.etf2l.org/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem "etf2l"
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install etf2l
```
